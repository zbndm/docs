# editPhoto
Ubah gambar profil obrolan.
## Parameter
```ts
ctx.telegram.editFoto(chatId,foto)
```

| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Obrolan/Grup/Channel yang akan diubah photo profilnya. |
| photo | `string` or `Buffer` | Ya | Lokasi file/Url/Buffer  |

## Contoh
```ts highlight="3"
bot.command("editPhoto", async (ctx)=>{
    if(!ctx.chat.private){
        let hasil = await ctx.telegram.editPhoto(ctx.chat.id,"https://tgsnake.js.org/images/tgsnake.jpg")
        console.log(hasil)
    }
})
```

| 400 | CHAT_NOT_MODIFIED | Pesan yang disematkan tidak diubah |

| 400 | PHOTO_CROP_SIZE_SMALL | Foto terlalu kecil |

| 400 | PHOTO_INVALID | Foto tidak valid |
| 400 | CHAT_ID_INVALID | Id obrolan yang diberikan tidak valid |
| 400 | CHAT_NOT_MODIFIED | Pesan yang disematkan tidak diubah |
