# Downloading file 
Downloading file with file id or [TypeMessageMediaDownload](/beta/modules/Utils.Medias.html#TypeMessageMediaDownload)

## Parameters 
```ts 
ctx.telegram.download(media,params)
```

| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| media | `string` or [TypeMessageMediaDownload](/beta/modules/Utils.Medias.html#TypeMessageMediaDownload) | true | media will be downloaded | 
| params |JSON object of [DownloadFileParams](/beta/interfaces/Interface.DownloadFileParams.html) | Optional | The download parameters will be used when downloading media | 

## Example 
```ts 
bot.on("message",async (ctx) => {
    if(ctx.media && ctx.media._ == "document"){
        let buffer = await ctx.telegram.download(media)
        return console.log(buffer)
    }
})
```

## Shorthand function 
```ts 
// you can also downloading the media like this
ctx.media.download(params)
```
