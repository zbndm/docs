# sendVenue
Sending media venue.
## Parameters 
```ts 
ctx.telegram.sendVenue(chatId,venue,more)
```

| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Chat/Groups/Channel id. |
| venue | JSON object of [Venue](#venue) or [MediaVenue](/beta/modules/Utils.Medias.html#MediaVenue) | true | Venue will be sending | 
| more | JSON Object of [defaultSendMediaMoreParams](/methods/Media/sendMedia#default-parameters) | Optional | More parametes will use to execute this function. | 

### Venue 

| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| latitude | number | true | Latitude |
| longitude | number | true | Longitude | 
| accuracyRadius | number | Optional | The estimated horizontal accuracy of the location, in meters; as defined by the sender. |
| title | string | true | Venue name | 
| address | string | true | Physical address of the venue | 
| provider | string | true | Venue provider: currently only "foursquare" needs to be supported | 
| id | string | true | Venue ID in the provider's database | 
| type | string | true | Venue Type in the provider's database | 

## Example 
```ts highlight="2-10"
bot.command("venue",async (ctx) => {
    let results = await ctx.telegram.sendVenue(ctx.chat.id,{
        latitude : 0,
        longitude : 0,
        title : "title",
        address : "address",
        provider : "provider",
        id : "some id here",
        type : "some type here"
    })
    console.log(results)
})
```
