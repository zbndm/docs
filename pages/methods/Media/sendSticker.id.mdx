# sendSticker
Mengirim Sticker dengan lokasi file/url/buffer/fileId/[MediaSticker](/beta/modules/Utils.Medias.html#MediaSticker). 
## Parameter
```ts
ctx.telegram.sendSticker(chatId,sticker)
```

| parameter | tipe | dibutuhkan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Chat/Groups/Channel id. |
| sticker | `string` or `Buffer` or [MediaSticker](/beta/modules/Utils.Medias.html#MediaSticker) | Ya | Sticker yang akan di kirimkan |

## Contoh
```ts highlight="3"
bot.on("message",async (ctx) => {
     if(ctx.media && ctx.media["_"] == "sticker"){
         let hasil = await ctx.telegram.sendSticker(ctx.chat.id,ctx.media.fileId)
         console.log(hasil)
     }
})
```
## Kemungkinan Kesalahan
Sama seperti [sendMedia](/methods/Media/sendMedia#possible-error)
