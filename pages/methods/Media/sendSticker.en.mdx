# sendSticker
Sending Sticker with file location/url/buffer/fileId/[MediaSticker](/beta/modules/Utils.Medias.html#MediaSticker). 
## Parameters 
```ts  
ctx.telegram.sendSticker(chatId,sticker)
```

| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Chat/Groups/Channel id. |
| sticker | `string` or `Buffer` or [MediaSticker](/beta/modules/Utils.Medias.html#MediaSticker) | true | Sticker will sending |

## Example 
```ts highlight="3"
bot.on("message",async (ctx) => {
    if(ctx.media && ctx.media["_"] == "sticker"){
        let results = await ctx.telegram.sendSticker(ctx.chat.id,ctx.media) 
        console.log(results)
    }
})
```
## Possible Error 
Same as [sendMedia](/methods/Media/sendMedia#possible-error)
