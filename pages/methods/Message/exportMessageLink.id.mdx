# exportMessageLink
Dapatkan tautan dan sematkan info pesan di saluran/grup super
## Parameter
```ts
ctx.telegram.exportMessageLink(chatId,messageId, more)
```
| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Chat/Groups/Channel id/username. |
| messageId | number | Ya | message id. | 
| more | JSON objek dari [Parameter Tambahan](#paramete-tambahan) | opsional | Parameter tambahan yang akan di gunakan untuk menjalankan fungsi ini. | 

### Parameter Tambahan
| params | type | description | 
| :----: | :---: | :---------: |
| thread | boolean | Apakah akan menyertakan juga ID utas, jika tersedia, di dalam tautan. | 
| grouped | boolean | Apakah akan menyertakan media lain yang dikelompokkan (untuk album) | 

## Contoh
```ts highlight="2"
bot.command("getLink", async (ctx) => {
    let hasil = await ctx.telegram.exportMessageLink(ctx.chat.id,ctx.id)
    return console.log(hasil)
})
```
## Kemungkinan Kesalahan
| kode | ketik | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid. |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini. |
| 400 | MESSAGE_ID_INVALID | ID pesan yang diberikan tidak valid. |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid. |
