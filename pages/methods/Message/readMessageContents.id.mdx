# readMessageContents
Memberi tahu pengirim tentang penerima yang telah mendengarkan pesan suara atau menonton video.
## Parameter
```ts
ctx.telegram.readMessageContents(messageId)
```
| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| messageId | Array&lt;number&gt; | Ya | List ID Pesan | 

## Contoh
```ts highlight="3"
bot.on("message",async (ctx)=>{
     if(ctx.media){
         let hasil = await ctx.telegram.readMessageContents([ctx.id])
         console.log(hasil)
     }
})
```
