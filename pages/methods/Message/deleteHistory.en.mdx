# deleteHistory
This method allow you to deletes communication history.
## Parameters 
```ts 
ctx.telegram.deleteHistory(chatId,more)
```
| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Target chat | 
| more | JSON object of [More Parameters](#more-parameters) | optional | More parametes will use to execute this function. | 

### More Parameters
| params | type | description | 
| :----: | :---: | :---------: |
| maxId | number | Maximum ID of message to delete. | 
| revoke | boolean | Whether to delete the message history for all chat participants. | 
| justClear | boolean | Just clear history for the current user, without actually removing messages for every chat user. | 

## Example 
```ts highlight="2"
bot.command("clear", async (ctx) => {
    let results = await ctx.telegram.deleteHistory(ctx.chat.id) 
    return console.log(results)
})
```
## Possible Error 
| code | type | description | 
| :---: | :---: | :------:| 
| 400 | CHAT_ID_INVALID | The provided chat id is invalid | 
| 400 | MESSAGE_ID_INVALID | The provided message id is invalid | 
| 400 | MSG_ID_INVALID | Invalid message ID provided | 
| 400 | PEER_ID_INVALID | The provided peer id is invalid |
