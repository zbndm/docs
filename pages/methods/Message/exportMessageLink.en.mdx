# exportMessageLink
Get link and embed info of a message in a channel/supergroup
## Parameters 
```ts 
ctx.telegram.exportMessageLink(chatId,messageId,more)
```
| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Chat/Groups/Channel id. |
| messageId | number | true | message id. | 
| more | JSON object of [More Parameters](#more-parameters) | optional | More parametes will use to execute this function. | 

### More Parameters 

| params | type | description | 
| :----: | :---: | :---------: |
| thread | boolean | Whether to also include a thread ID, if available, inside of the link. | 
| grouped | boolean | Whether to include other grouped media (for albums)

## Example 
```ts highlight="2"
bot.command("getLink", async (ctx) => {
    let results = await ctx.telegram.exportMessageLink(ctx.chat.id,ctx.id) 
    return console.log(results)
})
```
## Possible Error 
| code | type | description | 
| :---: | :---: | :------:| 
| 400 | CHANNEL_INVALID | The provided channel is invalid. | 
| 400 | CHANNEL_PRIVATE | You haven't joined this channel/supergroup. | 
| 400 | MESSAGE_ID_INVALID | The provided message id is invalid. | 
| 400 | MSG_ID_INVALID | Invalid message ID provided. |
