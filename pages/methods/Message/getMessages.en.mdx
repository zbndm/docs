# getMessages
Returns the list of messages by their IDs. 
## Parameters 
```ts 
ctx.telegram.getMessages(chatId,messageId,replies)
```
| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Chat/Groups/Channel id. |
| messageId | Array&lt;number&gt; | true | IDs of messages. |
| replies | boolean | Optional | if `true` it will getting the nested reply. and will making floodwait.

## Example 
```ts highlight="2"
bot.command("getMessages",async (ctx)=>{
    let results = await ctx.telegram.getMessages(ctx.chat.id,[ctx.id]) 
    console.log(results) 
})
```
