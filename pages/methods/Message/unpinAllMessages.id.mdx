## Ringkasan
Lepas sematan semua pesan dalam obrolan.
## Parameter
```ts
ctx.telegram.unpinAllMessages(chatId)
```
| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Target | 

## Contoh
```ts highlight="2"
bot.command("unpinAll",async (ctx)=>{
     let hasil = await ctx.telegram.unpinAllMessages(ctx.chat.id)
     console.log(hasil)
})
```
