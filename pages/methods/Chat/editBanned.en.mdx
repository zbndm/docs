# editBanned
Ban/unban/kick a user in a supergroup/channel.
## Parameters 
```ts 
ctx.telegram.editBanned(chatId,userId,more)
```

| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Target chat | 
| userId | `string` or `bigint` | true | Target user | 
| more | JSON object of [More Parameters](#more-parameters) | optional | More parametes will use to execute this function. | 

### More Parameters
| params | type | description | 
| :----: | :---: | :---------: |
| untilDate | number | Validity of said permissions (it is considered forever any value less then 30 seconds or more then 366 days). |
| viewMessages | boolean | If set, does not allow a user to view messages in a supergroup/channel/chat. | 
| sendMessages | boolean | If set, does not allow a user to send messages in a supergroup/chat. | 
| sendMedia | boolean | If set, does not allow a user to send any media in a supergroup/chat. |
| sendStickers | boolean | If set, does not allow a user to send stickers in a supergroup/chat. | 
| sendGifs | boolean | If set, does not allow a user to send gifs in a supergroup/chat. | 
| sendGames | boolean | If set, does not allow a user to send games in a supergroup/chat. | 
| sendInline | boolean | If set, does not allow a user to use inline bots in a supergroup/chat. | 
| sendPolls | boolean | If set, does not allow a user to send stickers in a supergroup/chat. |
| changeInfo | boolean | If set, does not allow any user to change the description of a supergroup/chat. | 
| inviteUsers | boolean | If set, does not allow any user to invite users in a supergroup/chat. | 
| pinMessages | boolean | If set, does not allow any user to pin messages in a supergroup/chat. | 
| embedLinks | boolean | If set, does not allow a user to embed links in the messages of a supergroup/chat. |

## Example 
```ts highlight="3"
bot.command("ban",async (ctx) => {
  if((!ctx.chat.private) && ctx.replyToMessage){
    let results = await ctx.telegram.editBanned(ctx.chat.id,ctx.replyToMessage.from.id) 
    console.log(results)
  }
})
```
## Shorthand function 
```ts 
// banChatMember 
ctx.telegram.banChatMember(chatId,userId)
// kickChatMember 
ctx.telegram.kickChatMember(chatId,userId)
// unbanChatMember 
ctx.telegram.unbanChatMember(chatId,userId)
// restrictChatMember 
ctx.telegram.restrictChatMember(chatId,userId,more)
```
## Possible Error 

| code | type | description | 
| :---: | :---: | :------:| 
| 400 | CHANNEL_INVALID | The provided channel is invalid | 
| 400 | CHANNEL_PRIVATE | You haven't joined this channel/supergroup | 
| 400 | CHAT_ADMIN_REQUIRED | You must be an admin in this chat to do this | 
| 403 | CHAT_WRITE_FORBIDDEN | You can't write in this chat | 
| 400 | MSG_ID_INVALID | Invalid message ID provided | 
| 400 | PINNED_DIALOGS_TOO_MUCH | Too many pinned dialogs |
| 400 | USER_ADMIN_INVALID | You're not an admin | 
| 400 | USER_ID_INVALID | The provided user ID is invalid |
