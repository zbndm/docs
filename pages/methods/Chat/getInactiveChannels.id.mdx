# getInactiveChannels
Dapatkan saluran dan grup yang tidak aktif.
## Parameter
```ts
ctx.telegram.getInactiveChannels()
```
Tidak ada parameter yang dibutuhkan.
## Contoh
```ts highlight="2"
bot.command("getInactiveChannels",async (ctx) => {
     let hasil = await ctx.telegram.getInactiveChannels()
     console.log(hasil)
})
```
## Kemungkinan Kesalahan
Tidak ada kemungkinan kesalahan.
## Bot tidak dapat menggunakan metode ini.
