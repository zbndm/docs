# getGroupsForDiscussion
Dapatkan semua grup yang dapat digunakan sebagai grup diskusi.
Obrolan grup lama yang dikembalikan harus ditingkatkan terlebih dahulu ke supergrup sebelum dapat ditetapkan sebagai grup diskusi.
Untuk menyetel supergrup yang dikembalikan sebagai grup diskusi, akses ke pesan lamanya harus diaktifkan menggunakan channels.togglePreHistoryHidden, terlebih dahulu.
## Parameter
```ts
ctx.telegram.getGroupsForDiscussion()
```
Tidak ada parameter yang dibutuhkan.
## Contoh
```ts highlight="2"
bot.command("getGroupsForDiscussion",async (ctx) => {
     let hasil = await ctx.telegram.getGroupsForDiscussion()
     console.log(hasil)
})
```
## Kemungkinan Kesalahan
Tidak ada kemungkinan kesalahan.
