# getInactiveChannels
Get inactive channels and supergroups.
## Parameters 
```ts 
ctx.telegram.getInactiveChannels()
```
No required parameters
## Example 
```ts highlight="2"
bot.command("getInactiveChannels",async (ctx) => {
    let results = await ctx.telegram.getInactiveChannels() 
    console.log(results)
})
```
