# editBanned
Melarang/membatalkan larangan/menendang pengguna di grup/saluran super.
## Parameters
```ts
ctx.telegram.editBanned(chatId,userId,more)
```
| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Chat/Groups/Channel id/username. |
| userd | `string` or `bigint` | Ya | User id/username. |
| more | JSON object dari [Parameter Tambahan](#parameter-tambahan) | opsional | Parameter tambahan yang akan di gunakan untuk menjalankan fungsi ini | 

### Parameter Tambahan
| parameter | tipe | deskripsi | 
| :----: | :---: | :---------: |
| untilDate | number | Validitas izin tersebut (dianggap selamanya nilai apa pun kurang dari 30 detik atau lebih dari 366 hari). | 
| viewMessages | boolean | Jika disetel, tidak mengizinkan pengguna melihat pesan di grup super/saluran/obrolan. | 
| sendMessage | boolean | Jika disetel, tidak mengizinkan pengguna mengirim pesan dalam supergrup/obrolan. | 
| sendMedia | boolean | Jika disetel, tidak mengizinkan pengguna mengirim media apa pun dalam supergrup/obrolan. | 
| sendStickers | boolean | Jika disetel, tidak mengizinkan pengguna mengirim stiker di supergrup/obrolan. | 
| sendGifs | boolean | Jika disetel, tidak mengizinkan pengguna mengirim gif di grup super/obrolan. | 
| sendGames | boolean | Jika disetel, tidak mengizinkan pengguna mengirim game dalam supergrup/obrolan. | 
| sendInline | boolean | Jika disetel, tidak mengizinkan pengguna untuk menggunakan bot sebaris dalam supergrup/obrolan. | 
| sendPolls | boolean | Jika disetel, tidak mengizinkan pengguna mengirim stiker di supergrup/obrolan. | 
| changeInfo | boolean | Jika disetel, tidak mengizinkan pengguna mengubah deskripsi supergrup/obrolan. | 
| inviteUsers | boolean | Jika disetel, tidak mengizinkan pengguna mana pun untuk mengundang pengguna di grup super/obrolan. | 
| pinMessages | boolean | Jika disetel, tidak mengizinkan pengguna mana pun untuk menyematkan pesan di grup super/obrolan. | 
| embedLinks | boolean | Jika disetel, tidak mengizinkan pengguna untuk menyematkan tautan dalam pesan grup super/obrolan. | 

## Contoh
```ts highlight="3"
bot.command("ban",async (ctx) => {
  if((!ctx.chat.private) && ctx.replyToMessage){
   let hasil = await ctx.telegram.editBanned(ctx.chat.id,ctx.replyToMessage.from.id)
    console.log(hasil)
  }
})
```

## Kemungkinan Kesalahan

| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini |
| 400 | CHAT_ADMIN_REQUIRED | Anda harus menjadi admin dalam obrolan ini untuk melakukan ini |
| 403 | CHAT_WRITE_FORBIDDEN | Anda tidak dapat menulis di obrolan ini |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid |
| 400 | PINNED_DIALOGS_TOO_MUCH | Terlalu banyak dialog yang dipasangi pin |
| 400 | USER_ADMIN_INVALID | Anda bukan admin |
| 400 | USER_ID_INVALID | ID pengguna yang diberikan tidak valid |
