# getChannels
Dapatkan info tentang saluran/supergrup.
## Parameter
```ts
ctx.telegram.getChannels(chatId)
```
| parameter | type | diperlukan | deskripsi |
| :----: | :---: | :-----: | :---------: |
| chatId | Array&lt;`string` atau `bigint`&gt; | Ya | Daftar id saluran. | 

## Contoh
```ts highlight="2"
bot.command("getChannels",async (ctx) => {
     let hasil = await ctx.telegram.getChannels([ctx.chat.id])
     console.log(hasil)
})
```
## Kemungkinan Kesalahan

| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid |
