# getFullChat
Mengembalikan info obrolan/saluran lengkap sesuai dengan ID-nya.
## Parameter
```ts
ctx.telegram.getFullChat(chatId)
```
| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Chat/Groups/Channel id/username. |

## Contoh
```ts highlight="2"
bot.command("getFullChat",async (ctx) => {
     let hasil = await ctx.telegram.getFullChat(ctx.chat.id)
})
```
## Kemungkinan Kesalahan

| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHAT_ID_INVALID | Id obrolan yang diberikan tidak valid |
| 400 PEER_ID_INVALID | ID rekan yang diberikan tidak valid |
| 400 CHANNEL_INVALID | Saluran yang disediakan tidak valid |
| 400 CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini |
| 403 CHANNEL_PUBLIC_GROUP_NA | saluran/supergrup tidak tersedia |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid
