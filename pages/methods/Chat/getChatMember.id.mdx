# getChatMember
Dapatkan info tentang peserta saluran/supergrup

## Parameter
```ts
ctx.telegram.getChatMember(chatId,userId)
```

| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Chat/Groups/Channel id/username. |
| userd | `string` or `bigint` | Ya | User id/username. |

## Contoh
```ts highlight="2"
bot.command("getChatMember",async (ctx) => {
    let hasil = await ctx.telegram.getParticipant(ctx.chat.id,ctx.from.id) // getChatMember dan getParticipant.adalah metode yang sama.
    console.log(hasil)
})
```
## Kemungkinan Kesalahan
| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid. |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini. |
| 400 | CHAT_ADMIN_REQUIRED | Anda harus menjadi admin dalam obrolan ini untuk melakukan ini. |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid. |
| 400 | PARTICIPANT_ID_INVALID | ID peserta yang ditentukan tidak valid. |
| 400 | USER_ID_INVALID | ID pengguna yang diberikan tidak valid. |
| 400 | USER_NOT_PARTICIPANT | Anda bukan anggota grup/saluran super ini |
