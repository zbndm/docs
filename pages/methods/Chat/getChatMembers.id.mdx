# getChatMembers
Mendapatkan daftar dari semua peserta di saluran atau obrolan.

## Parameter
```ts
ctx.telegram.getChatMembers(chatId,filters)
```

| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Chat/Groups/Channel id/username. |
| more | Objek JSON dari [Parameter Tambahan](#parameter-tambahan) | opsional | Parameter tambahan yang akan digunakan untuk menjalankan fungsi ini. |

### Parameter Tambahan
| parameter | tipe | deskripsi | 
| :----: | :---: | :---------: |
| offset | number | Dari indeks apa ke pagination dimulai. | 
| limit | number | Anggota daftar maksimal dalam pagination, defaultnya adalah `200` anggota.
| query | string | Kueri peserta berdasarkan nama | 
| filter | string | `all` Mendapatkan semua anggota. <br/> `kicked` Hanya mendapatkan anggota yang ditendang.<br/> `restricted` Hanya mendapatkan restricted anggota<br/> `bots` Hanya mendapatkan bot.<br/> `recents` Mendapatkan hanya anggota baru-baru ini.<br/> `administrator` Hanya mendapatkan administrator. | 

## Contoh
```ts highlight="2"
bot.command("getChatMembers",async (ctx) => {
    let result = await ctx.telegram.getParticipants(ctx.chat.id) // getChatMembers dan getParticipants adalah metode yang sama.
    console.log(hasil)
})
```
## Kemungkinan Kesalahan
| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid. |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini. |
| 400 | CHAT_ADMIN_REQUIRED | Anda harus menjadi admin dalam obrolan ini untuk melakukan ini. |
| 400 | CHAT_ID_INVALID | ID obrolan yang diberikan tidak valid. |
| 400 | PEER_ID_INVALID | ID rekan yang diberikan tidak valid |
