# editTitle
edit judul obrolan/grup/saluran.
## Parameter
```ts
ctx.telegram.editTitle(chatId,title)
```

| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | Target. |
| title | string | Ya | Judul baru | 

## Contoh
```ts highlight="3"
bot.command("editTitle",async (ctx) => {
    if(!ctx.chat.private){
        let hasil = await ctx.telegram.editTitle(ctx.chat.id,"hei judul baru")
        console.log(hasil)
    }
})
```
## Kemungkinan Kesalahan

| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid |
| 400 | CHAT_ADMIN_REQUIRED | Anda harus menjadi admin dalam obrolan ini untuk melakukan ini |
| 400 | CHAT_NOT_MODIFIED Pesan yang disematkan tidak diubah |
| 400 | CHAT_TITLE_EMPTY | Tidak ada judul obrolan yang disediakan |
| 403 | CHAT_WRITE_FORBIDDEN | Anda tidak dapat menulis di obrolan ini |
| 400 | CHAT_ID_INVALID | Id obrolan yang diberikan tidak valid |
| 400 | PEER_ID_INVALID | ID yang diberikan tidak valid |
