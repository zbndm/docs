# getChatMembers
Getting list from all participants in channel or chats. 

## Parameters
```ts 
ctx.telegram.getChatMembers(chatId,filters)
```

| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Target chat | 
| more | JSON object of [More Parameters](#more-parameters) | optional | More parametes will use to execute this function. | 

### More Parameters
| params | type | description | 
| :----: | :---: | :---------: |
| offset | number | From what index to pagination start. |
| limit | number | Max list members in pagination, default is `200` members. | 
| query | string | Query participants by name | 
| filter | string |`all` Getting all members.<br/> `kicked` Getting only kicked members.<br/> `restricted` Getting only restricted members<br/> `bots` Getting only bots.<br/> `recents` Getting only recent members.<br/> `administrators` Getting only administrators. | 

## Example 
```ts headlight="2"
bot.command("getChatMembers",async (ctx) => {
    let results = await ctx.telegram.getParticipants(ctx.chat.id) // getChatMembers and getParticipants is same methods.
    console.log(results)
})
```
## Possible Error 
| code | type | description | 
| :---: | :---: | :------:| 
| 400 | CHANNEL_INVALID | The provided channel is invalid. | 
| 400 | CHANNEL_PRIVATE | You haven't joined this channel/supergroup. | 
| 400 | CHAT_ADMIN_REQUIRED | You must be an admin in this chat to do this. | 
| 400 | CHAT_ID_INVALID | The provided chat id is invalid. | 
| 400 | PEER_ID_INVALID | The provided peer id is invalid |
