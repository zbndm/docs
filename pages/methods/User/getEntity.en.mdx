# getEntity
Getting chat or user info.
## Parameters 
```ts 
ctx.telegram.getEntity(chatId,useCache)
```

| params | type | required | description | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Chat/Groups/Channel id/username. |
| useCache | boolean | optional | Getting the entity from cache. Only work will numbers entity. Default is `true` |

## Example 
```ts highlight="2"
bot.command("me",async (ctx)=>{
    let results = await ctx.telegram.getEntity(ctx.chat.id,true) 
    console.log(results)
})
```
## Possible Error 

| code | type | description | 
| :---: | :---: | :------:| 
| 400 | Could not find the input entity | If you getEntity with id maybe it invalid. You can't find info about user with id if you've never seen that user. |
