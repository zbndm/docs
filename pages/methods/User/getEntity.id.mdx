# getEntity
Mendapatkan info obrolan atau pengguna.
## Parameters 
```ts 
ctx.telegram.getEntity(chatId,useCache)
```

| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | true | Chat/Groups/Channel id/username. |
| useCache | boolean | optional | Mendapatkan entitas dari cache. Hanya bekerja akan nomor entitas. Defaultnya adalah `true` |

## Contoh
```ts highlight="2"
bot.command("me",async (ctx)=>{
    let results = await ctx.telegram.getEntity(ctx.chat.id,true) 
    console.log(results)
})
```
## Kemungkinan Kesalahan

| kode | ketik | deskripsi | 
| :---: | :---: | :------:| 
| 400 | Could not find the input entity | Jika Anda mendapatkanEntity dengan id mungkin itu tidak valid. Anda tidak dapat menemukan info tentang pengguna dengan id jika Anda belum pernah melihat pengguna itu. |
