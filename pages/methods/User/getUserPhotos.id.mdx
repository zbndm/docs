# getUserPhotos 
Mendapatkan semua foto profil pengguna.

## Parameter

```ts
ctx.telegram.getUserPhotos(chatId, more)
```

| parameter | tipe | diperlukan | deskripsi | 
| :----: | :---: | :-----: | :---------: |
| chatId | `string` or `bigint` | Ya | User id/username |
| more | JSON object dari [Parameter Tambahan](#parameter-tambahan) | opsional | Parameter tambahan yang akan di gunakan untuk menjalankan fungsi ini | 

### Parameter Tam
| parameter | tipe | deskripsi | 
| :----: | :---: | :---------: |
| offset | number | Jumlah elemen daftar yang akan dilewati | 
| maxId | bigint | Jika nilai positif diberikan, metode ini hanya akan mengembalikan foto dengan ID kurang dari yang ditetapkan |
| limit | number | Jumlah elemen daftar yang akan dikembalikan | 

## Contoh
```ts highlight="2"
bot.command("getUserPhotos",async (ctx)=>{
     let hasil = await ctx.telegram.getUserPhotos(ctx.from.id)
     console.log(hasil)
})
```
## Kemungkinan Kesalahan

| kode | ketik | deskripsi |
| :---: | :---: | :------:|
| 400 | MAX_ID_INVALID | ID maks yang diberikan tidak valid |
| 400 | USER_ID_INVALID | ID pengguna yang diberikan tidak valid |
